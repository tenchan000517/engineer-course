## プロンプト設計ベース（ナレーション生成用）

### 共通制約（全カテゴリ）

```
あなたはYouTubeショートおよびTikTokで100万再生を連発する台本作家です。
以下の制約を厳守してください。
1. 尺は読み上げ速度でジャスト1分（文字数300〜350文字程度）。
2. 冒頭5秒で「常識の否定」や「強いフック」を入れる。
3. 抽象的な表現（「便利です」「凄いです」）は禁止。具体的な機能名や数字を使う。
4. 語尾は「〜です/ます」調と「〜だ/である」調を適度に混ぜ、テンポを良くする。
5. CTA（行動喚起）には「プロフリンク」は含めず、「コメント」や「保存」への誘導のみを行う。
```

### NG表現リスト

| NG表現 | 理由 | 代替案 |
|--------|------|--------|
| 「魔法」 | 抽象的すぎる | 具体的な機能名で説明 |
| 「〜を殺す」 | 過激すぎる | 「〜を超える」「〜に取って代わる」 |
| 英語直訳調 | 不自然な日本語 | 自然な口語に言い換え |
| 「便利です」「凄いです」 | 抽象的 | 具体的な数字・機能で示す |
| 「革命」「黒船」 | 使いすぎ注意 | 具体的な変化を描写 |

### カテゴリA（versus）プロンプト設計

```
## 命令書
以下の2つのツールを比較し、エンジニアや実務家が「どっちを使うべきか」迷っている現状に終止符を打つ、切れ味鋭いショート動画台本を作成してください。

## 比較対象
ツールA：[ツール名1]
ツールB：[ツール名2]

## 構成案
1. **フック (0-5秒)**: 「[ツールA] vs [ツールB]」という対立構造を提示し、「結論が出た」と言い切る。
2. **展開 (5-45秒)**:
   - 一方を「現在の王者（安定・実務向き）」、もう一方を「未来の革命児（尖っている・破壊的）」として対比させる。
   - それぞれの「具体的な機能名（例: Focus機能、Artifactsなど）」を必ず1つずつ出し、どんなメリットがあるか描写する。
   - 「どっちも良い」という逃げの結論は禁止。「こういう人はA、こういう人はB」と明確に線引きをする。
3. **CTA (45-60秒)**: 「あなたは安定派？革命派？」と問いかけ、コメント欄での議論を促す。

## 重要
「実務で使えるか否か」という視点を必ず盛り込んでください。
```

### カテゴリB（instant_hack）プロンプト設計

```
## 命令書
以下のツールを使い、既存の「無駄な手作業」を撲滅するためのショート動画台本を作成してください。

## 対象ツール
ツール名：[ツール名]
解決する課題：[具体的な悩み（例：ググる時間が長い、議事録作成が面倒）]

## 構成案
1. **フック (0-5秒)**: 視聴者が今やっている作業（例：手打ち入力、複数タブ検索）を「時間の無駄」と断定し、否定する。
2. **解決策 (5-45秒)**:
   - [ツール名]を使えば、その作業が「何秒」または「何クリック」で終わるか具体的に描写する。
   - ただの魔法として紹介せず、「なぜそれができるのか（機能名や仕組み）」を一言で解説し、納得感を持たせる。
   - 具体的なベネフィット（例：浮いた時間で早く帰れる、ミスがゼロになる）を提示する。
3. **CTA (45-60秒)**: 「手順を忘れないうちに保存推奨」という流れで締めくくる。

## 重要
視聴者が「うわ、私これ毎日やってる…」と自分事化できるような痛烈な課題提起を入れてください。
```

### カテゴリC（secret_feature）プロンプト設計

```
## 命令書
以下の「自律型/最新トレンド」ツールを紹介し、視聴者に「未来が来た」という衝撃を与える台本を作成してください。

## 対象ツール
ツール名：[ツール名]
主な凄さ：[特徴（例：勝手にブラウザ操作、動画のカメラ制御など）]

## 構成案
1. **フック (0-5秒)**: 従来のAI（チャット型など）を引き合いに出し、「それはもう古い」と切り捨てる。
2. **Wowファクター (5-45秒)**:
   - ツールが動いている様子が目に浮かぶような描写（例：勝手にマウスが動く、画像が一瞬で動画になる）を入れる。
   - 「人間が操作する」のではなく「AIが勝手に行動する」というパラダイムシフトを強調する。
   - 専門用語は使いすぎず、「要するに何ができるのか」を小学生でもわかる言葉で伝える。
3. **CTA (45-60秒)**: 「使ってみたい人はコメントで教えて」または「時代に乗り遅れたくない人は保存」と誘導する。

## 重要
「便利」という言葉は禁止。「魔法」「黒船」「革命」など、インパクトの強い言葉を選んでください。
```

### カテゴリD（warning）プロンプト設計

```
## 命令書
以下のツールやトレンドに対し、安易な利用に警鐘を鳴らす「辛口」な台本を作成してください。

## 対象ツール/トレンド
対象：[ツール名]
懸念点：[リスク（例：依存によるスキル低下、セキュリティ、課金など）]

## 構成案
1. **フック (0-5秒)**: 流行りのツールを礼賛する空気に水を差す。「警告します」「このままだとヤバいです」と強い言葉で止める。
2. **リスクの提示 (5-45秒)**:
   - 表面的なメリットの裏にある「致命的な落とし穴」を論理的に説明する。
   - 具体的な失敗シナリオ（例：エラーが直せなくなる、情報が漏れる）を提示し、視聴者の不安を喚起する。
   - 単なる批判で終わらせず、「賢い使い方はこうだ」というプロの視点を最後に少しだけ入れる。
3. **CTA (45-60秒)**: 「自戒も込めて保存してください」「周りの人にも教えてあげて」と締めくくる。

## 重要
「使うな」と否定するのではなく、「思考停止で使うな」というニュアンスで、リテラシーの重要性を説いてください。
```

### カテゴリE（ranking）プロンプト設計

```
## 命令書
以下の5つのAIツールをランキング形式（またはまとめ形式）で紹介する、疾走感のある台本を作成してください。

## ランキング内容
5位：[ツール名]（特徴一言）
4位：[ツール名]（特徴一言）
3位：[ツール名]（特徴一言）
2位：[ツール名]（特徴一言）
1位：[ツール名]（特徴一言）

## 構成案
1. **フック (0-5秒)**: 「今月話題の神AIツールTOP5」「これを知らないと損する」と宣言し、期待感を高める。
2. **カウントダウン (5-50秒)**:
   - 5位から順に紹介する。1つのツールにつき「名前」＋「最強のメリット一言（ワンビッグメッセージ）」のみを詰め込む。
   - 余計な説明は省き、テンポ（BGMに合うようなリズム）を最優先する。
   - 1位に向けて徐々にテンションやツールの重要度を上げていく。
3. **CTA (50-60秒)**: 「全部覚えるのは無理だと思うので、保存して後で見返してください」と誘導する。

## 重要
1ツールあたり7〜8秒で切ってください。「〜ができて、〜もできて」というダラダラした説明は厳禁です。
```

### 設計メモ（プロンプト関連）

- **用途**: ナレーション生成専用（コンテンツ部分は別途設計）
- **n8n対応**: `[ツール名]`等のプレースホルダーはn8n変数（`{{ $json.tools }}`等）に置き換え
- **JSON出力**: 別途出力形式を追加してn8n仕様に対応させる

## 改良版プロンプト（Gemini 1.5 Flash対応・検討中）

**改善ポイント**:
- 「創造」させない: 型（テンプレート）に情報を当てはめる作業に徹させ、モデル性能によるブレをなくす
- NGワードの厳格化: 「魔法」「殺す」「直訳調」をシステム的に排除
- JSON項目の強制割り当て: どの情報をどこで使うかを番号で指定

**注意**: 禁止用語が大々的に入っているため、優秀な推論を妨げる可能性あり。確定ではなく検討中。
