# Next.js ポートフォリオサイト制作講座

## 📖 講座概要

この講座では、**Claude Code**と**Next.js**を使って、動画も含めた本格的なポートフォリオサイトを作成する方法を学びます。非エンジニアの初心者でも、順を追って学習することで、プロフェッショナルなWebサイトを自分で構築・運用できるようになります。

---

## 🎯 学習目標（KGI）

この講座を修了すると、以下のことができるようになります：

✅ **動画も含めたポートフォリオサイトがNext.jsで作れるようになる**
✅ **UIを自分で改善・カスタマイズできるようになる**
✅ サイトを公開し、継続的に更新できるようになる

## 💡 この講座の特徴

### Claude Codeを活用
従来のプログラミング講座と異なり、**Claude Code**というAIアシスタントを活用します。コーディングの細かい構文を暗記する必要はありません。重要なのは「何を作りたいか」を言語化する力です。

### 実践重視
架空のサンプルではなく、**あなた自身のポートフォリオサイト**を作ります。講座終了時には、実際に公開できる成果物が手に入ります。

### UI改善スキルに注力
単にサイトを作るだけでなく、**自分でデザインを改善できる力**を養います。モジュール6では、ベンチマークサイトの分析から、自分なりのカスタマイズまでを学びます。
