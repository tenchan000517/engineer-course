# 入力データ
タイトル: {{ $json.title }}

リサーチポイント: {{ $json.research_points }}

# 最重要指示（絶対厳守）

1. 入力データの「タイトル」「比較ツール」は一切変更せず、そのまま使用すること
2. ツール名やモデル名が不明・最新でも、入力されたものをそのまま出力に使うこと
3. 入力データを「古い」「存在しない」「間違い」と判断して別の名前に置き換えることは絶対禁止
4. Web検索やリサーチは行わないこと。入力データのみを使用すること
5. ベンチマークスコアや具体的な数値は生成禁止。機能の特徴のみを説明すること

# 禁止事項（厳守）
- 入力データのツール名を変更・修正すること
- 想像や推測で数値・スコアを生成すること
- 絵文字の使用（一切禁止）
- 抽象的な表現（「便利」「凄い」「魔法」など）
- 過激な表現（「〜を殺す」「〜は終わり」など）

# ナレーション制約
1. 合計300〜350文字（読み上げ60秒相当）
2. 冒頭5秒で視聴者の関心を引くフック
3. 両ツールの具体的な機能名を含める（入力のresearch_pointsを活用）
4. 「こういう人はA、こういう人はB」と明確に線引き
5. CTAは「コメント」「保存」への誘導のみ（プロフリンク禁止）

# スライド制約
- 各セットは3行、各行は全角14文字以内（厳守）
- 英数字は半角2文字=全角1文字で換算
- 3行は対になる要素か同じ属性で揃える
- ツール名は入力データのまま正確に記載

# サムネイル制約
- thumb_main: 8文字以内（疑問形または煽り）
- thumb_sub: 6文字以内×3行（改行区切り）

# キャプション制約
- 絵文字禁止
- 親しみやすい文体（〜ですよね、〜なんです）
- 見出し（▶）で3セクションに分ける
- 箇条書き（①②③）でポイント整理

# 出力形式（JSON）
以下のJSON形式で出力してください。他の文字は含めないでください。

- Set 1 (0:00-0:05): フック - 比較テーマの提示
